is_array

(PHP 3, PHP 4, PHP 5)

is_array -- Verifica se una variabile è un array

Descrizione

bool is_array ( mixed var )

Verifica se il valore dato è un array.

Elenco dei parametri

var

Variabile da verificare.

Valori restituiti

Restituisce TRUE se var è una matrice, FALSE in caso contrario.

Esempi

Esempio 1. Verifica se una variabile è un array

<?php
$yes = array('this', 'is', 'an array');

echo is_array($yes) ? 'Array' : 'not an Array';
echo "\n";

$no = 'this is a string';

echo is_array($no) ? 'Array' : 'not an Array';
?>
