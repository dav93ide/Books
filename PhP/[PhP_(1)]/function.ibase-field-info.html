ibase_field_info

(PHP 3 >= 3.0.7, PHP 4, PHP 5)

ibase_field_info --  Ottiene informazioni su un campo

Descrizione

array ibase_field_info ( int result, int field number )

Restituisce un array con informazioni relative a un campo dopo che una query select Ã¨ stata eseguita. L'array ha la forma name, alias, relation, length e type.

$rs=ibase_query("SELECT * FROM tablename");
$coln = ibase_num_fields($rs);
for ($i=0; $i < $coln; $i++) {
    $col_info = ibase_field_info($rs, $i);
    echo "name: ".$col_info['name']."\n";
    echo "alias: ".$col_info['alias']."\n";
    echo "relation: ".$col_info['relation']."\n";
    echo "length: ".$col_info['length']."\n";
    echo "type: ".$col_info['type']."\n";
    }
