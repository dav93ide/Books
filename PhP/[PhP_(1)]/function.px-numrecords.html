px_numrecords

(PECL)

px_numrecords -- Returns number of records in a database

Descrizione

int px_numrecords ( resource pxdoc )

Get the number of records in a database file.

Valori restituiti

Returns the number of records in a database file. The return value of this function is identical to the element numrecords in the associated array returned by px_get_info().
