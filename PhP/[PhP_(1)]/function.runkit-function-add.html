runkit_function_add

(PECL)

runkit_function_add --  Add a new function, similar to create_function()

Descrizione

bool runkit_function_add ( string funcname, string arglist, string code )

Elenco dei parametri

funcname

Name of function to be created

arglist

Comma separated argument list

code

Code making up the function

Valori restituiti

Restituisce TRUE in caso di successo, FALSE in caso di fallimento.

Esempi

Esempio 1. A runkit_function_add() example

<?php
create_function('testme','$a,$b','echo "The value of a is $a\n"; echo "The value of b is $b\n";');
testme(1,2);
?>
