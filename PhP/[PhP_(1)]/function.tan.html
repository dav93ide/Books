tan

(PHP 3, PHP 4, PHP 5)

tan -- Tangente

Descrizione

float tan ( float arg )

La funzione tan() restituisce la tangente del parametro arg. Il parametro arg deve essere espresso in radianti.

Esempio 1. Esempio per tan()

<?php

echo tan(M_PI_2); // 1

?>

Vedere anche: atan(), sin(), cos() e deg2rad().
