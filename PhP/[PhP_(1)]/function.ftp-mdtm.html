ftp_mdtm

(PHP 3 >= 3.0.13, PHP 4, PHP 5)

ftp_mdtm -- Restituisce l'orario dell'ultima modifica al file corrente

Descrizione

int ftp_mdtm ( resource ftp_stream, string remote_file )

La funzione ftp_mdtm() controlla l'orario dell'ultima modifica ad un file, e lo restituisce in formato timestamp Unix. Se si verifica un errore, o il file non esiste, restituisce -1.

Restituisce un timestamp Unix in caso di successo, o -1 in caso di errore.

Esempio 1. Esempio di funzione ftp_mdtm()

<?php

$file = 'somefile.txt';

// stabilisce la connessione
$conn_id = ftp_connect($ftp_server);

// si collega con nome utente e password
$login_result = ftp_login($conn_id, $ftp_user_name, $ftp_user_pass);

// recupera l'ultimo orario di modifica
$buff = ftp_mdtm($conn_id, $file);

if ($buff != -1) {
    // somefile.txt e' stato modificato l'ultima volta il: 26 marzo 2003 14:16:41.
    echo "$file e' stato modificato l'ultima volta il : " . date("F d Y H:i:s.", $buff);
} else {
    echo "Impossibile visualizzare mdtime";
}

// chiude la connessione
ftp_close($conn_id);

?>

Nota: Non tutti i server includono questa funzione!

Nota: La funzione ftp_mdtm() non funziona con le directories.
