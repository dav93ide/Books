nl2br

(PHP 3, PHP 4, PHP 5)

nl2br --  Inserisce il tag HTML di 'a capo' prima di tutti i caratteri di 'a capo' della stringa

Descrizione

string nl2br ( string string )

Restituisce la stringa string con '<br />' inserito prima di tutti i newline.

Nota: A partire dal PHP 4.0.5, nl2br() Ã¨ conforme a XHTML. Tutte le versioni precedenti alla 4.0.5 restituiscono la string con '<br>' invece di '<br />' inserito prima degli 'a capo'.

Esempio 1. Esempio di uso di nl2br()

<?php
echo nl2br("foo isn't\n bar");
?>

this will output : 

foo isn't<br />
 bar

Vedere anche htmlspecialchars(), htmlentities(), wordwrap() e str_replace().
