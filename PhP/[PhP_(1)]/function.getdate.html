getdate

(PHP 3, PHP 4, PHP 5)

getdate -- Riceve informazioni su data/orario

Descrizione

array getdate ( [int timestamp] )

Restituisce un array associativo contenente le informazioni sulla data del timestamp, o dell'attuale orario locale se non Ã¨ stato assegnato timestamp, con i seguenti elementi di array:

  • "seconds" - secondi

  • "minutes" - minuti

  • "hours" - ore

  • "mday" - giorno del mese

  • "wday" - giorno della settimana, numerico : da 0 come Domenica a 6 come Sabato

  • "mon" - mese, numerico

  • "year" - anno, numerico

  • "yday" - giorno dell'anno, numerico; i.e. "299"

  • "weekday" - giorno della settimana, testuale, per intero; i.e. "Friday"

  • "month" - mese, testuale, per intero; i.e. "January"

Esempio 1. Esempio di getdate()

$today = getdate();
$month = $today['month'];
$mday = $today['mday'];
$year = $today['year'];
echo "$month $mday, $year";
