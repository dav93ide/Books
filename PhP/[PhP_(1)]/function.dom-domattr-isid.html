DOMAttr->isId() --  Verifica se l'attributo Ã¨ un ID definito

Descrizione

class DOMAttr {

bool isId ( void )

}

Questa funzione verifica se l'attributo sia un ID definito.

In base allo standard DOM, questa funzione riochiede la presenza di una DTD che definisca l'attributo ID di tipo ID. Occorre validare il documento con DOMDocument->validate() oppure DOMDocument::validateOnParse prima di utilizzare questa funzione.

Valori restituiti

Restituisce TRUE in caso di successo, FALSE in caso di fallimento.

Esempi

Esempio 1. DOMAttr->isId() Example

<?php

$doc = new DomDocument;

// Occorre validare il documento prima di puntare all'id
$doc->validateOnParse = true;
$doc->Load('book.xml');

// Recuperiamo l'attributo chianat id del capitolo documento
$attr = $doc->getElementsByTagName('chapter')->item(0)->getAttributeNode('id');

var_dump($attr->isId()); // bool(true)

?>
