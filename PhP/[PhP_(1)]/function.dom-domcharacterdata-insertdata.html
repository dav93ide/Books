DOMCharacterData->insertData()

(no version information, might be only in CVS)

DOMCharacterData->insertData() --  Inserisce una stringa all'offset di 16 bit indicato

Descrizione

class DOMCharacterData {

void insertData ( int offset, string data )

}

Inserisce la stringa data alla posizione offset.

Elenco dei parametri

offset

L'offset del carattere da cui cominciare ad inserire.

data

La stringa da inserire.

Valori restituiti

No value is returned.

Eccezioni

DOM_INDEX_SIZE_ERR

Generato se offset è negativo o se è maggiore di un numero di 16 bit.
