DOMElement->getAttributeNode() -- Restituisce il nodo attributo

Descrizione

class DOMElement {

DOMAttr getAttributeNode ( string name )

}

Restituisce il nodo attributo con nome name per l'elemento corrente.

Elenco dei parametri

name

Nome dell'attributo.

Valori restituiti

Il nodo attributo.
