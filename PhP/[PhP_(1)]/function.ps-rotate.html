ps_rotate

(PECL)

ps_rotate -- Sets rotation factor

Descrizione

bool ps_rotate ( resource psdoc, float rot )

Sets the rotation of the coordinate system.

Valori restituiti

Restituisce TRUE in caso di successo, FALSE in caso di fallimento.
