DomElement->get_elements_by_tagname

(no version information, might be only in CVS)

DomElement->get_elements_by_tagname --  Restituisce l'elemento con il tag dato

Descrizione

array DomElement->get_elements_by_tagname ( string nome )

Questa funzione restituisce un array con tutti gli elementi che hanno come nome del tag nome. Ogni elemento dell'array Ã¨ un oggetto DomElement.

Esempio 1. Ottenere il contenuto

<?php
if (!$dom = domxml_open_mem($xmlstr)) {
  echo "Error while parsing the document\n";
  exit;
}

$root = $dom->document_element();

$node_array = $root->get_elements_by_tagname("element");

for ($i = 0; $i<count($node_array); $i++) {
    $node = $node_array[$i];
    echo ("The element[$i] is: ".$node->get_content());
}

?>
