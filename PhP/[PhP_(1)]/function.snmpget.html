(PHP 3, PHP 4, PHP 5)

snmpget -- Preleva un oggetto SNMP

Descrizione

string snmpget ( string hostname, string community, string object_id [, int timeout [, int retries]] )

La funzione restituisce il valore di un oggetto SNMP se ha successo, altrimenti FALSE se si verificano errori

La funzione snmpget(), viene utilizzata per leggere il valore dell'oggetto SNMP specificato da object_id. L'agente SNMP a cui accedere viene specificato nel parametro hostname, mentre la comunit√† viene indicata in community.
