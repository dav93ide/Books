DOMElement->getAttributeNodeNS() --  Restituisce un nodo attributo

Descrizione

class DOMElement {

DOMAttr getAttributeNodeNS ( string namespaceURI, string localName )

}

Restituisce il nodo attributo cone nome localName nello spazio dei nomi namespaceURI per il nodo corrente.

Elenco dei parametri

namespaceURI

URI dello spazio dei nomi.

localName

Nome locale.

Valori restituiti

The attribute node.
