DOMNode->removeChild() --  Rimuove un nodo figlio dalla lista dei nodi figli

Descrizione

class DOMNode {

DOMNode removeChild ( DOMNode oldnode )

}

Questa funzione rimuove un nodo figlio dalla lista dei nodi figli.

Elenco dei parametri

oldnode

Il nodo rimosso.

Valori restituiti

Se si riesce a rimuovere il nodo figlio, la funzione restituisce il vecchio nodo figlio.

Eccezioni

DOM_NO_MODIFICATION_ALLOWED_ERR

Generato se il nodo è in sola lettura.

DOM_NOT_FOUND

Generato se oldnode non è un figlio di questo nodo.

Esempi

Nel seguente esempio si rimuoverà il nodo 'chapter' dal nostro documento XML.

Esempio 1. Rimozione di un nodo figlio

<?php

$doc = new DOMDocument;
$doc->load('book.xml');

$book = $doc->documentElement;

// Cerchiamo il nodo e lo rimuoviamo
$chapter = $book->getElementsByTagName('chapter')->item(0);
$oldchapter = $book->removeChild($chapter);

echo $doc->saveXML();
?>
