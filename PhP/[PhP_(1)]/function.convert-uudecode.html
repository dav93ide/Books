convert_uudecode

(PHP 5)

convert_uudecode --  Decodifica una stringa codificata con uuencode

Descrizione

string convert_uudecode ( string data )

La funzione convert_uudecode() decodifica una stringa codificata con uuencode.

Esempio 1. Esempio di uso di convert_uudecode()

<?php
/* Can you imagine what this will print? :) */
echo convert_uudecode("+22!L;W9E(%!(4\"$`\n`");
?>

Vedere anche convert_uuencode().
