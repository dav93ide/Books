ps_closepath

(PECL)

ps_closepath -- Closes path

Descrizione

bool ps_closepath ( resource psdoc )

Connects the last point with first point of a path.

Valori restituiti

Restituisce TRUE in caso di successo, FALSE in caso di fallimento.
