(PHP 4 >= 4.0.4, PHP 5)

gmp_powm -- Modulo di un elevamento a potenza.

Descrizione

resource gmp_powm ( resource base, resource exp, resource mod )

Calcola il (base elevato a potenza exp) modulo mod. Se exp è negativo, il risultato sarà indefinito.
