str_shuffle

(PHP 4 >= 4.3.0, PHP 5)

str_shuffle -- Mischia una stringa in modo casuale

Descrizione

string str_shuffle ( string str )

str_shuffle() mischia una stringa. Viene generata una di tute le possibili permutazioni.

Esempio 1. Esempio di uso di str_shuffle()

<?php
$str = 'abcdef';
$shuffled = str_shuffle($str);

// This will echo something like: bfdaec
echo $shuffled;
?>

Vedere anche: shuffle() e rand().
