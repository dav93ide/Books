DOMDocument->load() --  Carica un documento XML da un file

Descrizione

class DOMDocument {

bool load ( string filename )

}

Carica un documento XML da un file.

Questo metodo può essere richiamato staticamente per caricare e creare un oggetto DOMDocument. L'inbvocazione statica può essere utilizzata quando non occorre impostare le proprietà di DOMDocument prima del caricamento.

Elenco dei parametri

filename

Il percorso al documento XML.

Valori restituiti

Restituisce TRUE in caso di successo, FALSE in caso di fallimento.

Esempi

Esempio 1. Creazione di un documento

<?php
$doc = DOMDocument::load('book.xml');
echo $doc->saveXML();

$doc = new DOMDocument();
$doc->load('book.xml');
echo $doc->saveXML();
?>
