tanh

(PHP 4 >= 4.1.0, PHP 5)

tanh -- Tangente iperbolica

Descrizione

float tanh ( float arg )

Restituisce la tangente iperbolica di arg, definita come sinh(arg)/cosh(arg).

Vedere anche: tan(), atanh(), sin() e cos().
