gmp_strval

(PHP 4 >= 4.0.4, PHP 5)

gmp_strval -- Converte un numero GMP in una stringa

Descrizione

string gmp_strval ( resource gmpnumber [, int base] )

Converte un numero GMP in una stringa rappresentato in base base. La base di default Ã¨ 10. Le basi consentite variano dal 2 al 36.

Esempio 1. Converte un numero GMP in una stringa

<?php
    $a = gmp_init("0x41682179fbf5");
    printf ("Decimal: %s, 36-based: %s", gmp_strval($a), gmp_strval($a,36));
?>
