ps_scale

(PECL)

ps_scale -- Sets scaling factor

Descrizione

bool ps_scale ( resource psdoc, float x, float y )

Sets horizontal and vertical scaling of the coordinate system.

Valori restituiti

Restituisce TRUE in caso di successo, FALSE in caso di fallimento.
