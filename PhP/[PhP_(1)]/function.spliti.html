spliti

(PHP 4 >= 4.0.1, PHP 5)

spliti --  Suddivide una stringa in una matrice usando le espressioni regolari senza distinguere tra maiuscole e minuscole

Descrizione

array spliti ( string pattern, string string [, int limit] )

Questa funzione ha un comportamento identico a split() tranne che per il fatto di non distinguere tra lettere maiuscole e minuscole. characters.

Il seguente esempio suddivide una stringa utilizzando la lettera 'a' come separatore:

Esempio 1. Esempio di uso di spliti()

<?php
$string = "aBBBaCCCADDDaEEEaGGGA";
$chunks = spliti ("a", $string, 5);
print_r($chunks);
?>
