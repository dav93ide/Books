SQLiteResult->prev -- Si posiziona sulla riga precedente di un set di risultati

Descrizione

bool sqlite_prev ( resource result )

Versione ad oggetti (metodo):

}

sqlite_prev() si sposta alla riga precedente di result.

Elenco dei parametri

result

Risorsa risultato di SQLite. Questo parametro non Ã¨ richiesto quando di usa la versione ad oggetti.

Nota: This function cannot be used with unbuffered result handles.

Valori restituiti

La funzione restituisce TRUE se ha successo, oppure FALSE se non esiste un record precedente.
