is_resource

(PHP 4, PHP 5)

is_resource --  Verifica se una variabile è una risorsa

Descrizione

bool is_resource ( mixed var )

Verifica se una data variabile sia di tipo risorsa.

Elenco dei parametri

var

La variabile da valutare.

Valori restituiti

Restituisce TRUE se la variabile individuata dal parametro var è di tipo resource, altrimenti restituisce FALSE.

Esempi

Esempio 1. Esempio di utilizzo di is_resource()

<?php

$db_link = @mysql_connect('localhost', 'mysql_user', 'mysql_pass');
if (!is_resource($db_link)) {
    die('Can\'t connect : ' . mysql_error());
}

?>
