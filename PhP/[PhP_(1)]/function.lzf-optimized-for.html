lzf_optimized_for

(PECL)

lzf_optimized_for --  Determina il tipo di ottimizzazione del modulo LZF.

Descrizione

int lzf_optimized_for ( void )

Restituisce 1 se LZF è ottimizzato per la velocità, 0 per la compressione.
