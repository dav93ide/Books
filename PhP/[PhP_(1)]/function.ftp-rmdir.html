ftp_rmdir

(PHP 3 >= 3.0.13, PHP 4, PHP 5)

ftp_rmdir -- Elimina una directory

Descrizione

bool ftp_rmdir ( resource ftp_stream, string directory )

Elimina la directory specificata. directory deve essere un percorso assoluto o relativo ad una directory vuota.

Restituisce TRUE in caso di successo, FALSE in caso di fallimento.

Esempio 1. Esempio di funzione ftp_rmdir()

<?php

$dir = 'www/';

// stabilisce la connesione
$conn_id = ftp_connect($ftp_server);

// si collega con username and password
$login_result = ftp_login($conn_id, $ftp_user_name, $ftp_user_pass);

// prova a eliminare la directory $dir
if (ftp_rmdir($conn_id, $dir)) {
    echo "Directory $dir eliminata\n";
} else {
    echo "Si e' verificato un problema durante l'eliminazione di $dir\n";
}

ftp_close($conn_id);

?>
