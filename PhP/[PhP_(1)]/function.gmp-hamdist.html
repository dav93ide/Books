gmp_hamdist

(PHP 4 >= 4.0.4, PHP 5)

gmp_hamdist -- Distanza dell'hamming

Descrizione

int gmp_hamdist ( resource a, resource b )

Restituisce la distanza di hamming tra a e b. Entrambe gli operandi dovrebbero essere non-negativi.
