cpdf_set_creator

(PHP 3 >= 3.0.8, PHP 4, PHP 5 <= 5.0.4)

cpdf_set_creator -- Imposta il campo creatore nel documento pdf

Descrizione

bool cpdf_set_creator ( int pdf_document, string creatore )

La funzione cpdf_set_creator() imposta il creatore del documento pdf. Restituisce TRUE in caso di successo, FALSE in caso di fallimento.

Vedere anche: cpdf_set_subject(), cpdf_set_title() e cpdf_set_keywords().
