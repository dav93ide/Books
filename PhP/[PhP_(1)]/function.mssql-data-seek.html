mssql_data_seek

(PHP 3, PHP 4, PHP 5)

mssql_data_seek -- Sposta il puntatore di riga interno

Descrizione

bool mssql_data_seek ( resource id_risultato, int numero_riga )

Restituisce TRUE in caso di successo, FALSE in caso di fallimento.

La funzione mssql_data_seek() sposta il puntatore di riga, interno al risultato associato all'identificativo di risultato, alla riga indicata dall'argomento numero_riga, la prima riga inzia con il numero 0. La chiamata successiva a mssql_fetch_row() restituir√† la riga richiesta.
