cpdf_setgray_fill

(PHP 3 >= 3.0.8, PHP 4, PHP 5 <= 5.0.4)

cpdf_setgray_fill -- Imposta il grigio come colore per il riempimento

Descrizione

bool cpdf_setgray_fill ( int documento_pdf, float valore )

La funzione cpdf_setgray_fill() imposta il valore corrente di grigio come riempimento del path. Restituisce TRUE in caso di successo, FALSE in caso di fallimento.

Vedere anche cpdf_setrgbcolor_fill().
