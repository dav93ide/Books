pg_options

(PHP 3, PHP 4, PHP 5)

pg_options -- Estrae le opzioni associate alla connessione

Descrizione

string pg_options ( resource connessione )

pg_options() restituisce una stringa contenente le opzioni specificate alla risorsa PostgreSQL connessione.
