DOMDocument->__construct() --  Crea un nuovo oggetto DOMDocument

Descrizione

class DOMDocument {

__construct ( [string version [, string encoding]] )

}

Crea un nuovo oggetto DOMDocument.

Elenco dei parametri

version

Numero di versione del documento come parte della dichiarazione XML.

encoding

Codifica del docuemnto utilizzata nella dichisrazione XML.

Esempi

Esempio 1. Esempio di creazione di un nuovo DOMDocument

<?php

$dom = new DOMDocument('1.0', 'iso-8859-1');

echo $dom->saveXML(); /* <?xml version="1.0" encoding="iso-8859-1"?> */

?>
