fdf_get_status

fdf_get_status -- Restituisce il valore per la chiave /STATUS

Descrizione

string fdf_get_status ( resource fdf_document )

La funzione fdf_get_status() restituisce il valore per la chiave /STATUS.

Vedere anche fdf_set_status().
