DOMDocument->save() --  Scarica l'albero XML interno in un file

Descrizione

class DOMDocument {

mixed save ( string filename )

}

Crea un docuemnto XML dalla rappresentazione DOM. Usualmente questa funzione Ã¨ chiamata dopo la creazione da zero di un nuovo documento DOM come nell'esempio seguente.

Elenco dei parametri

filename

Percorso del file in cui scrivere il documento XML.

Valori restituiti

La funzione restituisce il numero di byte scritti, oppure FALSE in caso di errore.

Esempi

Esempio 1. Salvataggio di un albero DOM in un file

<?php

$doc = new DOMDocument('1.0');
// output formattato
$doc->formatOutput = true;

$root = $doc->createElement('book');
$root = $doc->appendChild($root);

$title = $doc->createElement('title');
$title = $root->appendChild($title);

$text = $doc->createTextNode('This is the title');
$text = $title->appendChild($text);

echo 'Wrote: ' . $doc->save("/tmp/test.xml") . ' bytes'; // Wrote: 72 bytes

?>
