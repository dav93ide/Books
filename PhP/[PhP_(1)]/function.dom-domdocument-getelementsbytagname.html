DOMDocument->getElementsByTagName()

(no version information, might be only in CVS)

DOMDocument->getElementsByTagName() -- Cerca tutti gli eleemnti per un dato nome di tag

Descrizione

class DOMDocument {

DOMNodeList getElementsByTagName ( string name )

}

Questa funzione crea una nuova istanza della classe DOMNodeList contenente gli elementi con il nome tag cercato.

Elenco dei parametri

name

Il nome del tag da cercare. Il valore spaciale * indica tutti i tag.

Valori restituiti

Una nuova istanza dell'oggetto DOMNodeList contenente tutti gli elementi trovati.
