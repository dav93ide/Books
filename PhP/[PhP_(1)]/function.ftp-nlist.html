ftp_nlist

(PHP 3 >= 3.0.13, PHP 4, PHP 5)

ftp_nlist -- Restituisce la lista dei file nella directory prescelta

Descrizione

array ftp_nlist ( resource ftp_stream, string directory )

Restituisce un array di nomi di file dalla directory specificata in caso di corretta esecuzione oppure FALSE in caso di errore.

Esempio 1. Esempio di funzione ftp_nlist()

<?php

// stabilisce la connessione
$conn_id = ftp_connect($ftp_server);

// si collega con nome utente e password
$login_result = ftp_login($conn_id, $ftp_user_name, $ftp_user_pass);

// recupera il contenuto della directory corrente
$contents = ftp_nlist($conn_id, ".");

// visualizza $contents
var_dump($contents);

?>
