ps_end_page

(PECL)

ps_end_page -- End a page

Descrizione

bool ps_end_page ( resource psdoc )

Ends a page which was started with ps_begin_page().

Valori restituiti

Restituisce TRUE in caso di successo, FALSE in caso di fallimento.

Vedere anche:

ps_begin_page()
