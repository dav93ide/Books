pdf_rotate

(PHP 3 >= 3.0.6, PHP 4, PECL)

pdf_rotate -- Sets rotation

Description

bool pdf_rotate ( resource pdfdoc, float phi )

Rotate the coordinate system by phi degrees. Restituisce TRUE in caso di successo, FALSE in caso di fallimento.
