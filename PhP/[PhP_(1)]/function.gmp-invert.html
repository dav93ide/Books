gmp_invert

(PHP 4 >= 4.0.4, PHP 5)

gmp_invert -- Inversione di modulo

Descrizione

resource gmp_invert ( resource a, resource b )

Calcola l'inverso di a modulo b. Restituisce un valore FALSE se l'inversione non esiste.
