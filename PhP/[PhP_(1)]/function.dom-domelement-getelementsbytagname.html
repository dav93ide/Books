DOMElement->getElementsByTagName() -- Restituisce gli elementi in base al nome del tag

Descrizione

class DOMElement {

DOMNodeList getElementsByTagName ( string name )

}

Questa funzione restituisce una nuova istanza della classe DOMNodeList contenente gli elementi i cui nomi di tag corrispondono al parametro name, nell'ordine in cui sono incontrati durante l'attraversamento dell'albero.

Elenco dei parametri

name

Il nome del tag. Utilizzare "*" per indicare di restituire tutti gli elementi del documento.

Valori restituiti

Questa funzione crea una nuova istanza della classe DOMNodeList contenente tutti gli elementi trovati.
