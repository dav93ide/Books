filepro_fieldwidth

(PHP 3, PHP 4, PHP 5)

filepro_fieldwidth -- Restituisce la dimensione di un campo

Descrizione

int filepro_fieldwidth ( int field_number )

Restituisce la lunghezza del campo riferito all'indice inserito come parametro field_number.
