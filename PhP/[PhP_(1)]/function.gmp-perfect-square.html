gmp_perfect_square

(PHP 4 >= 4.0.4, PHP 5)

gmp_perfect_square -- Controllo quadrato perfetto

Descrizione

bool gmp_perfect_square ( resource a )

Restituisce un valore vero TRUE se a è un quadrato perfetto,falso FALSE se non lo è.

Vedere anche: gmp_sqrt(), gmp_sqrtrm().
