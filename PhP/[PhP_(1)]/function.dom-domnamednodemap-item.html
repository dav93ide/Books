DOMNamedNodeMap->item() -- Recupera un nodo tramite il suo indice

Descrizione

class DOMNamedNodeMap {

DOMNode item ( int index )

}

Recupera un nodo tramite l'indice index all'interno dell'oggetto DOMNamedNodeMap.

Elenco dei parametri

index

Indice all'interno della tabella.

Valori restituiti

Il nodo alla posizione indexnella tabella, oppure NULL se non si tratta di un indice valido (maggiore o uguale al numero dei nodi in questa tabella).
