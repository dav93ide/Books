gmp_pow

(PHP 4 >= 4.0.4, PHP 5)

gmp_pow -- Eleva un numero a potenza

Descrizione

resource gmp_pow ( resource base, int exp )

Eleva la base base ad una potenza exp. Nel caso di 0^0 il risultato sarà 1. L'argomento exp non può essere negativo.
