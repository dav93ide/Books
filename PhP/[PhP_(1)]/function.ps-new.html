ps_new

(PECL)

ps_new -- Creates a new PostScript document object

Descrizione

resource ps_new ( void )

Creates a new document instance. It does not create the file on disk or in memory. It just sets up everything.

Valori restituiti

Resource of PostScript document or FALSE on failure.
