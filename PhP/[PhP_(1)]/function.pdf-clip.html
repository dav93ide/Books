pdf_clip

(PHP 3 >= 3.0.6, PHP 4, PECL)

pdf_clip -- Clips to current path

Description

bool pdf_clip ( resource pdfdoc )

Use the current path as clipping path. Restituisce TRUE in caso di successo, FALSE in caso di fallimento.
