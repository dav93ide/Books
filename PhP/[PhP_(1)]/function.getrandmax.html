getrandmax

(PHP 3, PHP 4, PHP 5)

getrandmax -- Mostra il più grande numero casuale disponibile

Descrizione

int getrandmax ( void )

Restituisce il valore massimo che può essere restituito da una chiamata alla funzione rand().

Vedere anche rand(), srand() e mt_getrandmax().
