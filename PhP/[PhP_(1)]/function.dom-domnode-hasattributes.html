DOMNode->hasAttributes() --  Verifica se un nodo ha degli attributi

Descrizione

class DOMNode {

bool hasAttributes ( void )

}

Questa funzione verifica se un nodo ha degli attributi. Il nodo verificato deve essere di tipo XML_ELEMENT_NODE.

Valori restituiti

Restituisce TRUE in caso di successo, FALSE in caso di fallimento.
