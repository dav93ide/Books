pdf_delete

(PHP 4 >= 4.0.5, PECL)

pdf_delete -- Deletes a PDF object

Description

bool pdf_delete ( resource pdfdoc )

Delete the PDF resource, and free all internal resources. Restituisce TRUE in caso di successo, FALSE in caso di fallimento.

See also pdf_new().
