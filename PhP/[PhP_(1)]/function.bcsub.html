bcsub

(PHP 3, PHP 4, PHP 5)

bcsub --  Sottrae un numero a precisione arbitraria da un altro

Descrizione

string bcsub ( string primo operando, string secondo operando [, int precisione] )

Sottrae il primo operando dal secondo operando e retituisce il risultato in una stringa. Il parametro opzionale scale Ã¨ usato per impostare il numero di cifre dopo il punto decimale nel risultato.

Esempi

Esempio 1. esempio di bcsub()

<?php

$a = 1.234;
$b = 5;

echo bcsub($a, $b);     // -3
echo bcsub($a, $b, 4);  // -3.7660

?>
