closedir

(PHP 3, PHP 4, PHP 5)

closedir -- Chiude l'handle della directory

Descrizione

void closedir ( resource dir_handle )

Chiude il flusso directory indicato da dir_handle. Il flusso deve essere stato aperto in precedenza da opendir().
