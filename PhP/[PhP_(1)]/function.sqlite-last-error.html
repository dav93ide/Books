sqlite_last_error

(PHP 5)

sqlite_last_error

(no version information, might be only in CVS)

SQLiteDatabase->lastError -- Restituisce il codice di errore dell'ultimo errore accorso sul database

Descrizione

int sqlite_last_error ( resource dbhandle )

Versione ad oggetti (metodo):

class SQLiteDatabase {

int lastError ( void )

}

Restituisce il codice di errore dall'ultima operazione eseguita sul database dbhandle. Una descrizione interpretabile dell'errore può essere ricavata tramite la funzione sqlite_error_string().

Elenco dei parametri

dbhandle

Risorsa SQLite Database restituita da sqlite_open () quando usato in modo procedurale. Questo parametro non è richiesto nel metodo ad oggetti.

Vedere anche:

sqlite_error_string()
