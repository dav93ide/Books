vprintf -- Visualizza una stringa formattata

Descrizione

int vprintf ( string format, array args )

Visualizza i valori di una matrice come stringa formattata in base a format (che Ã¨ descritto nella documentazione di sprintf()).

Restituisce la lunghezza della stringa prodotta.
