DOMElement->setAttribute() -- Aggiunge un nuovo attributo

Descrizione

class DOMElement {

bool setAttribute ( string name, string value )

}

Imposta l'attributo con nome name al dato valore. Se l'attributo non esiste lo crea.

Elenco dei parametri

name

Nome dell'attributo.

value

Valore dell'attributo.

Valori restituiti

Restituisce TRUE in caso di successo, FALSE in caso di fallimento.

Eccezioni

DOM_NO_MODIFICATION_ALLOWED_ERR

Generato se il nodo Ã¨ in sola lettura.

Esempi

Esempio 1. Setting an attribute

<?php
$doc = new DOMDocument("1.0");
$node = $doc->createElement("para");
$newnode = $doc->appendChild($node);
$newnode->setAttribute("align", "left");
?>
