gmp_setbit

(PHP 4 >= 4.0.4, PHP 5)

gmp_setbit -- Imposta bit

Descrizione

resource gmp_setbit ( resource &a, int index [, bool set_clear] )

Sets bit index in a. set_clear definisce se il bit è settato su 0 o su 1. Di default il bit è settato a 1.
