ftp_site

(PHP 3 >= 3.0.15, PHP 4, PHP 5)

ftp_site -- Invia al server un comando SITE

Descrizione

bool ftp_site ( resource ftp_stream, string cmd )

La funzione ftp_site() invia al server FTP il comando specificato dal parametro cmd. I comandi SITE non sono standardizzati e sono diversi da server a server. They are Sono utili per operare, ad esempio, sui permessi dei file o sull'appartenenza a gruppi.

Esempio 1. Invio di un comando SITE ad un server ftp

<?php
/* Connessione al server FTP */
$conn = ftp_connect('ftp.example.com');
if (!$conn) die('Impossibile connettersi a ftp.example.com');

/* Collegarsi come "user" con password "pass" */
if (!ftp_login($conn, 'user', 'pass')) die('Errore nel collegamento come "user" a ftp.example.com');

/* Invio: comando "SITE CHMOD 0600 /home/user/privatefile" al server ftp */
if (ftp_site($conn, 'CHMOD 0600 /home/user/privatefile')) {
   echo "Comando eseguito correttamente.\n";
} else {
   die('Errore nell'esecuzione del comando.');
}
?>

Restituisce TRUE in caso di successo, FALSE in caso di fallimento.

Vedere anche: ftp_raw()
