DOMNamedNodeMap->getNamedItem() --  Recupera un nodo indicandone il nome

Descrizione

class DOMNamedNodeMap {

DOMNode getNamedItem ( string name )

}

Recupera il nodo indicato da nodeName.

Elenco dei parametri

name

Il nome del nodo da cercare.

Valori restituiti

Il nodo (di qualsiasi tipo) con il nome richiesto, oppure NULL se non viene trovato.
