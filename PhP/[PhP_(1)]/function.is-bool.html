is_bool

(PHP 4, PHP 5)

is_bool --  Verifica se una variabile è di tipo boolean

Descrizione

bool is_bool ( mixed var )

Verifica se la variabile data sia di tipo boolean

Elenco dei parametri

var

La variabile da valutare.

Valori restituiti

Restituisce TRUE se var è di tipo boolean, FALSE in caso contrario.

Esempi

Esempio 1. Esempi di is_bool()

<?php
$a = false;
$b = 0;

// Poichè $a è booleana, questa è vera
if (is_bool($a)) {
    echo "Yes, this is a boolean";
}

// Poichè $b non è booleana, questa non è vera
if (is_bool($b)) {
    echo "Yes, this is a boolean";
}
?>
