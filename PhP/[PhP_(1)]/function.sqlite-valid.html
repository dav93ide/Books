Verifica se una o più righe sono disponibili nel dato set di risultati.

Elenco dei parametri

result

Risorsa risultato di SQLite. Questo parametro non è richiesto quando la funzione viene usata ad oggetti.

Nota: This function cannot be used with unbuffered result handles.

Valori restituiti

Restituisce TRUE se vi sono più righe disponibili da result, oppure FALSE in caso contrario.
