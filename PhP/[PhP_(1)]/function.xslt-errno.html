xslt_errno

(PHP 4 >= 4.0.3, PECL)

xslt_errno -- Restituisce un numero di errore

Descrizione

int xslt_errno ( resource xh )

Restituisce un codice di errore che descrive l'ultimo errore successo sul processore XSLT indicato.

Vedere anche:

xslt_error().
